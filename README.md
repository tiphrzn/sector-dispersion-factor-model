# Modèle de Dispersion sectorielle sur le S&P 500
Ce projet de fin d'étude au sein de la Gestion sous Mandat analyse le rôle de la dispersion des secteurs sur le marché des actions et son utilisation potentiel dans la gestion des risques/rendements de nos différents portefeuilles multi-assets.

Pour calculer la dispersion, j'ai utilisé deux méthodes:
  -CSSD (Cross Sectional Standard Deviation) : très sensible aux valeurs extrêmes (utiles pour les chocs de marché : Crise Covid, Crise des subprimes...)
  -CSAD (Cross-sectional Absolute Deviation): Plus robuste, préférée pour éviter que la dispersion soit uniquement dérivée par un seul secteur aberrant
